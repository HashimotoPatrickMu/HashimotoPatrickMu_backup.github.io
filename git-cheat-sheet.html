Git Command Summary

Local Git Configs

# set-up local config in the folder
git config user.name "hashimoto"
git config user.email "your.email@example.com"
# check
git config --list
git config user.name
git config user.email

Pushing

# push to origin branch
git push origin $BRANCH
# set upstream branch, creates a tracking reference for every branch, allows to use commands such as `git pull` without any arguments
git push -u origin $BRANCH

Undoing Commits

# soft reset (undo commit but keep changes), ~$STEPS$ how many steps want to go back, check by `git log`
git reset --soft HEAD~1
# hard reset (undo commit and discard changes)
git reset --hard HEAD~1
git push origin <branch-name> --force
# `git revert`, `git restore`, `git rm`

Merging Branches

# branch of merging into
git checkout $BRANCH-0
# branch of merging from
git merge $BRANCH-1

Synchronizing with Remote Repository

# automatically fetches and merges changes from the remote branch to current branch.
git pull origin $BRANCH
# fetches changes from all branches without merging, allows to review changes before manually merging them into a specific branch
git fetch origin
git merge origin/$BRANCH
